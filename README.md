# ShopHub - React E-Commerce Project

A simple e-commerce web application built with **React**, **TypeScript**, and **Tailwind CSS**. The project demonstrates key frontend concepts including state management, routing, responsive design, and a fully functional shopping cart.

## Deployment

You can view a live demo of the project here: (https://app.netlify.com/projects/shophup-project/deploys)

## Feature

- **Shopping Cart**: Add, remove, and update product quantities with state management.
- **Order Summary**: View a breakdown of selected products with total price calculation.
- **Checkout Page**: Clears the cart after confirming the order.
- **Responsive Layout**: Optimized for mobile and desktop screens.
- **Contact Page**: Form with client-side validation using React and TypeScript.

## Technologies Used

- **React** (with Hooks and functional components)
- **TypeScript** for type safety
- **Tailwind CSS** for styling and responsive layouts
- **React Router** for page navigation
- **React Context API** for cart state management

## Installation

1. Clone the repository:

```bash
git clone https://github.com/NoroffFEU/jsfw-2025-v1-morten_js_frameworks_ca.git
cd YOUR_REPO
```

2. Install dependencies:

- npm install

3. Start the development server

- npm start

## Usage

- Navigate between pages using the header links.
- Add products to the cart from the Home page.
- Adjust quantities or remove products in the Cart page.
- Proceed to checkout to clear the cart and view the order confirmation.
- Use the Contact page to send messages (with form validation).
